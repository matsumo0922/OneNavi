# TTS フレーズから逆算するナビゲーション必要機能一覧

ドライブサポーターの TTS フレーズ (768件) を網羅的に分析し、各フレーズを発話するために必要なナビゲーション機能を逆算した資料。

- 対象: NAVITIME ドライブサポーター v8.3.1
- 作成日: 2026-04-11
- 元データ: `drive-supporter-tts-guide.md` (全768フレーズ)

## 1. 基本ターンバイターン案内

### 関連フレーズ

| フレーズID | 内容 |
|---|---|
| #3-18 | 距離「...先」形式 (およそ50m先 〜 およそ10km先) |
| #32-39 | 方向・連結用 (直進方向、/ 斜め右方向、/ 右方向、/ ...) |
| #40-47 | 方向・文末用 (直進です。/ 斜め右方向です。/ 右方向です。/ ...) |
| #76-79 | タイミング修飾 (この先 / その先 / まもなく / 更に) |
| #689 | 直進方向です。 |
| #754-755 | 2つ目/3つ目の信号を |
| #757-758 | 次の信号を / この信号を |
| #764-766 | 右折後 / 左折後 / 交差点を通過後 |

### 必要機能

| 機能 | 説明 |
|---|---|
| ルートマッチング | GPS位置をルート上にスナップし、現在位置を特定する |
| ガイドポイント管理 | ルート上の全案内地点（交差点・分岐等）のリストと距離計算 |
| 方向判定 (8方向) | 各ガイドポイントでの進行方向を8方向で分類する。直進 / 斜め右 / 右 / 右手前 / 戻る / 左手前 / 左 / 斜め左 |
| 距離段階判定 | 案内地点までの距離を離散値 (50m / 100m / 200m / 300m / 400m / 500m / 600m / 700m / 800m / 900m / 1km / 2km / 3km / 4km / 5km / 10km) に丸める |
| タイミング制御 | 「まもなく」「この先」「その先」「更に」の使い分けロジック。案内地点までの距離・直前の案内状態に応じて切り替える |
| 信号機データ参照 | 交差点に信号があるかの情報。「この信号を」と「交差点を」の出し分けに必要 |
| N番目信号カウント | 「2つ目/3つ目の信号を」のために前方の信号機を数える |
| 連続案内判定 | 次の案内地点が近い場合、読点止め方向 (#32-39) →「その先」で繋ぐ。複数 PhraseData の連結判定 |
| 右左折後案内 | 右折後 / 左折後 / 交差点通過後に次の案内を即座に出す |

### 組み立てパターン

```
[基本] [距離] + [対象物] + [方向(文末)]
例: #6(およそ300m先) + #758(この信号を) + #42(右方向です。)

[連続] [距離] + [方向(連結)] → [タイミング] + [方向(文末)]
例: #78(まもなく) + #34(右方向、) → #77(その先) + #59(分岐です。)

[N番目] [距離] + [N番目の信号] + [方向(文末)]
例: #5(およそ200m先) + #754(2つ目の信号を) + #46(左方向です。)

[通過後] [通過後修飾] + [方向(文末)]
例: #764(右折後) + #46(左方向です。)
```

# 02. Functional & Non-Functional Requirements

## Functional Requirements

### Must Have

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-01 | 起動 | Intent share 起動 | Google Maps 等から座標/地名を共有して OneNavi を起動し、ルート案内に進める |
| F-02 | 検索 | 目的地・経由地検索 | 住所、施設名、座標による検索。ナビ中の経由地追加にも対応する |
| F-03 | ルート案内 | ターンバイターン案内 | 交差点名、方向、距離、推定到着時刻を表示する |
| F-04 | ルート案内 | リルート | 経路逸脱時に外部ナビ API ライブラリを通じて再検索する |
| F-05 | ルート案内 | 複数ルート提案 | ルート選択時に複数候補を表示する |
| F-06 | 音声案内 | 自然な日本語音声案内 | 外部ナビ API の案内情報をもとに、Cloud TTS または Android TTS で読み上げる |
| F-07 | 高速パネル | IC/JCT/SA/PA 一覧表示 | 通過時刻・距離付きで一覧表示。一般道でも右左折リストを表示する |
| F-08 | 車線案内 | 曲がるべき車線の表示 | 主要道路・高速道路での車線インジケーターを表示する |
| F-09 | 地図表示 | リアルタイム渋滞情報の表示 | 道路ごとの渋滞レベルを色分け表示する |
| F-10 | 地図表示 | ルートライン描画 | 渋滞色を反映したルートラインを Google Maps 上に描画する |

### Should Have

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-11 | 交差点拡大図 | 高速 JCT / 交差点イメージ | 外部ナビ API ライブラリが返す画像を優先して表示する |
| F-12 | 方面看板 | 高速道路の方面看板表示 | 外部ナビ API ライブラリの案内情報を UI に反映する |
| F-13 | 料金表示 | 高速料金 | Google Routes API で補完する |
| F-14 | Android Auto | ナビ画面対応 | 公開 API で地図 surface と案内 UI を成立させる |

### Nice to Have

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-15 | 車種切替 | 自動車 / 二輪車プロファイル | 交通規制の違いを反映したルーティングを検討する |
| F-16 | オフライン | オフライン時の最低限動作 | 取得済みルート・案内情報の表示継続を検討する |
| F-17 | 速度警告 | 制限速度超過時の警告 | 外部ナビ API ライブラリまたは将来の補助データで検討する |
| F-18 | 燃費計算 | 高速パネルからの燃費計算 | 距離・時刻データから逆算する |

## Non-Functional Requirements

| ID | カテゴリ | 要件 |
|---|---|---|
| NF-01 | ライセンス | OSS として公開できる構成にする |
| NF-02 | 配布 | 外部ナビ API の認証情報は repo に含めない |
| NF-03 | UI/UX | Google Maps レベルの視認性とスムーズなアニメーションを目標にする |
| NF-04 | パフォーマンス | 60fps での地図描画、音声案内の遅延抑制 |
| NF-05 | コスト | 個人利用で無料枠または自己負担範囲に収まること |
| NF-06 | 言語 | Kotlin、Jetpack Compose |
| NF-07 | 最小 API レベル | Android Auto と Google Maps SDK の要件に準拠する |

# 02. Functional & Non-Functional Requirements

## Functional Requirements

### Must Have (Phase 1)

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-01 | 起動 | Intent share 起動 | Google Maps 等から座標/地名を共有して OneNavi を起動し、即ルート案内開始 |
| F-02 | 検索 | 最低限の目的地・経由地検索 | 住所、施設名、座標による検索。ナビ中の経由地追加に対応 |
| F-03 | ルート案内 | ターンバイターン案内 | 交差点名、方向、距離、推定到着時刻を表示 |
| F-04 | ルート案内 | リルート | 経路逸脱時の自動リルート |
| F-05 | ルート案内 | 複数ルート提案 | ルート選択時に複数候補を表示、ナビ中も代替ルートを地図上に表示 |
| F-06 | 音声案内 | 自然な日本語音声案内 | 通称使用（「環八」等）、信号機案内（「2つ目の信号を左です」）、自然な言い回し（「この信号を右折です」） |
| F-07 | 高速パネル | IC/JCT/SA/PA 一覧表示 | 通過時刻・距離付きで一覧表示。一般道でも右左折リスト表示 |
| F-08 | 車線案内 | 曲がるべき車線の表示 | 主要道路・高速道路での車線インジケーター |
| F-09 | Android Auto | 完全対応 | アプリ版と同等機能。Auto 実行中もスマホ側でアプリ操作可能 |
| F-10 | 地図表示 | リアルタイム渋滞情報の表示 | 道路ごとの渋滞レベルを色分け表示 |
| F-11 | 地図表示 | ルートライン描画 | 渋滞色を反映したルートライン |

### Should Have (Phase 2)

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-12 | 車種切替 | 自動車 / 二輪車プロファイル | 交通規制の違い（二輪車通行禁止等）を反映したルーティング |
| F-13 | 交差点拡大図 | 高速 JCT の分岐イラスト | 道路基盤地図情報 + OSM データからプロシージャル生成 |
| F-14 | 方面看板 | 高速道路の方面看板表示 | Mapbox banner データ + OSM destination タグから生成 |
| F-15 | 料金表示 | 高速料金（ETC 割引含む） | Google Routes API で補完 |

### Nice to Have (Phase 3+)

| ID | カテゴリ | 要件 | 詳細 |
|---|---|---|---|
| F-16 | オフライン | オフラインナビ | Mapbox オフラインマップ + Valhalla 自前ホスト |
| F-17 | 速度警告 | 制限速度超過時の警告 | Mapbox maxspeed データを活用 |
| F-18 | 燃費計算 | 高速パネルからの燃費計算 | 距離・時刻データから逆算 |

## Non-Functional Requirements

| ID | カテゴリ | 要件 |
|---|---|---|
| NF-01 | ライセンス | OSS として公開（ライセンス未定） |
| NF-02 | 配布 | 各ユーザーが自分の API キーを設定して使用 |
| NF-03 | UI/UX | Google Maps レベルの美しさ・視認性・スムーズなアニメーション |
| NF-04 | パフォーマンス | 60fps での地図描画、音声案内の遅延なし |
| NF-05 | コスト | 個人利用で無料枠内に収まること |
| NF-06 | 言語 | Kotlin、Jetpack Compose |
| NF-07 | 最小 API レベル | 未定（Android Auto の要件に準拠） |
